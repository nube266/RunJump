# RunJump
## 概要
横スクロールの2Dアクションゲームです。  
全3面で各ステージの最後にいるボスを倒すとステージクリアです。

## ゲームの遊び方
WebGL版を以下のように公開しています。  
[遊びたい方はこちら](https://nube266.github.io/run_jump_webgl/)

## 推奨環境(動作確認をした環境)
- OS: macOS Big Sur  
- ブラウザ: Safari  
- 画面を最大化すること推奨  
- 音声が再生されない場合は[Safari]→[このWebサイトでの設定]→[自動再生: すべてのメディアを自動再生]  

## 開発環境
OS: macOS  
Unityのバージョン: 2018.3.0f2
